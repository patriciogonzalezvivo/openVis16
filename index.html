This is a single shader. They are atomic and self contain. One **single main function*** execute in parallel for each pixel. Is the Sauron ring of GPUs. It have some build in functions like: ```gl_FragCoord``` (read only variable that provides the position of the pixel on the viewport) and ```gl_FragColor``` (write only variable that contain the color of the pixel); Then it have some variables that we can "send" from the CPU that are going to be uniform to all the threats call ```uniforms```. There we are passing the resolution (or dimensions) of the viewport, the mouse position and elipse time.
